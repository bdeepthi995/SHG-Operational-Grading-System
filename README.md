# SHG Operational Grading System

# Project Overview
The SHG Operational Grading System is a Java-based console application designed to evaluate and grade Self Help Groups (SHGs) based on their operational performance.  
The grading is calculated using meeting participation and loan repayment behavior with time-based evaluation. This project simulates a real-world rural finance grading process used by banks and NGOs.

# Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- Multithreading
- JSON (as a database)
- Eclipse IDE
